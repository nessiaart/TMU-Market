# TMU Marketplace

TMU Marketplace is a full-stack web application designed for the Toronto Metropolitan University community! Built using modern technologies and industry development practices, the platform enables students to buy, sell, and connect within an admin moderated TMU exclusive app. The application was originally developed by our team in 2024. 

---

## Features

**Video Demo:** https://drive.google.com/file/d/1AVUaNmL8HYjoEWbRFhskosxuyR3xxiCi/view
- Secure user authentication with encrypted passwords
- Create, edit, and delete marketplace listings
- Browse posts with search and filtering
- Real-time chat between buyers and sellers
- User profile and account management
- Admin moderation tools for platform safety
- Online/offline user status indicators within real-time chat
- Games page and trivia API for entertainment 

---

## Tech Stack
**Frontend**
- React
- JavaScript
- Tailwind CSS + Bootstrap
- Axios
- Socket.io Client

**Backend**
- Node.js

- Express.js

- Socket.io

- bcrypt (password hashing)

- MongoDB

---

## Installation

1. Clone the repository 

2. Install dependencies in  `/server`, `/socket` and `/frontend_react`

3. Run npm start in `/server`, `/socket` and `/frontend_react`
